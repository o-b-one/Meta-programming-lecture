# Meta-programming-lecture

## Install

`npm i`

## Descriptors

For run the descriptors example execute

    npm run descriptors

## Proxy

For run the proxy example execute

    npm run proxy

## Decorator

For run the decorators example execute

    npm run decorators


## SugoiJS

For run the [SugoiJS](http://www.sugoijs.com) example execute

    npm run sugoi
